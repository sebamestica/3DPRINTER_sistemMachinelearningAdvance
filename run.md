# Comando para Iniciar el Proyecto

Para ejecutar la aplicación correctamente desde PowerShell o cualquier terminal con los permisos necesarios, utiliza el siguiente bloque de comandos. Este comando asegura que te sitúas en el directorio del frontend antes de lanzar el servidor de desarrollo Vite.

```powershell
cd "c:\dev\PLA_3dPrinter_RESISTENCE\frontend"; npm run dev
```

### Notas Técnicas
- **Puerto por defecto**: http://localhost:5173
- **Navegación**:
  - `/` → Decision Studio (Panel Principal 5x5x5)
  - `/advanced` → Advanced Lab (Inspección Técnica)
- **Modos de Visualización**: Prueba los botones de la izquierda para cambiar entre visualización sólida, translúcida y solo estructura.
