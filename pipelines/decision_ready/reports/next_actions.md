# Recomendaciones de Siguientes Acciones

1. **Prioridad 1**: Vincular `specimen_id` con el `FDM_Dataset` para obtener parametros de fabricacion real.
2. **Ampliacion**: El modelo actual captura la geometria pero no el proceso. Se recomienda duplicar el dataset con mas variabilidad de temperatura.
3. **Modelo**: Con mas de 100 muestras, se puede evaluar redes neuronales o ensambles mas complejos.
4. **Optimización**: No tiene sentido optimizar hiperparametros por encima de lo actual hasta ganar en volumen muestral.
